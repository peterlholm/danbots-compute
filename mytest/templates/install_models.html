Installation of new models


  1. Copy the new models to server folder /var/www/danbots/compute/data/nnmodels
  2. Rename the old model files
  3. Rename the new model files to H_model.h5 and L_model.h5
  4. Restart the webserver ($sudo systemctl restart apache2.service)

file transfer

The files can be copied to the server with:
  • $ rcp <local_file> <user>@sal2.local:/<destignation>
